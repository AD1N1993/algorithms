# Algorithms Lessons

Репозиторий организован по урокам. Каждый урок — отдельная папка в `src/`.

## Структура

- `src/lesson-00-byte-practice/` - базовая практика с битами на `byte`
- `src/lesson-01-bit-basics/` - операции с битами и `IntBitVector`
- `src/lesson-02-bit-insert/` - вставка битового диапазона из `B` в `A`

## Как запускать

Открой Demo-файл нужного урока и нажми Run в IDE.
